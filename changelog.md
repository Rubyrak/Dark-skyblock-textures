# v1.0 - (2026-02-18)
## Fixes
- Bumped resource pack version to support future updates
- Added more blocks to support the new end revamp from alpha
## New content (Respackopts support)

Respackots its a mod that provides resource packs a config menu, after installing respackopts, a menu button will appear besides all supported resource packs. **All these new options are disabled by default and will only work if Respackopts is installed**, so you can use the pack as usual if you don't download it; in addition, all areas can be toggled individually with this mod.

### Extra zones
- Crimson Isle (Designed for Tarantula and Blaze slayers, and somehow resembles the LoL death filter)
- Dungeons (Prone to change)

### Custom textures
- Entity fire removal (Removes the fire of flaming spiders and blazes)
- Fire (Supports multiple colors)
- Lava (Supports multiple colors)

### Future plans
- We're open to suggestion for new zones and different textures to currently existing features.
- Current zones in development:
  - Jerry's island
  - Mushroom desert
